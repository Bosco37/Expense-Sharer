# Expense Sharer App

A full-stack expense sharing application built using React and Node.js.

## Live Demo
Frontend: https://expense-frontend-6hbg.onrender.com  
Backend: https://expense-sharer.onrender.com  

## Features
- Add members
- Add expenses
- Track debts
- View history

## Tech Stack
- React.js
- Node.js
- Express.js
- Axios
- Render (Deployment)
